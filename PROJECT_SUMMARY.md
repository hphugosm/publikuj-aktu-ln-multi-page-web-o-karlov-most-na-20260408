# Project Summary

Goal: Publikuj aktuální multi-page web o Karlově mostě na GitHub Pages. Zachovej strukturu, manifest i delivery pack, použij existující GitHub publish flow a vygeneruj PUBLISH_REPORT.md. Neřeš obsahovou factual kvalitu, pouze publish/deploy/online dostupnost.
Task type: portfolio-page

## Co bylo vytvořeno
- Statický web připravený pro publish přes GitHub Pages.
- Exportní delivery pack pro prezentaci a předání.
- Prezentační režim se zaměřením na jasnou strukturu a navigaci.

## Struktura webu
- index.html (Domů)
- history.html (Historie)
- statues.html (Sochy)
- sources.html (Zdroje)
- faq.html (FAQ)
- project-update.html (Aktualizace)

## Shared assets
- assets/styles.css

## Hodnota projektu
- Vzniká konzistentní multi-page web, který jde bezpečně iterovat a opakovaně publikovat.
- Manifest + update mode snižují riziko chaotických změn mezi iteracemi.
